Ques 2. Explain br tags?

br tages are referred as line break tag and it does not require any close tag. This used to provide a line break between the two content lines or tages whenever it is used.





Ques 3. How many different types of list we have in HTML?

We do have 2 types of lists in HTML. They are Ordered lists and unordered lists. Ordered lists are referred with the tag called ol type. It has several sub categories in it like Number(1), Upper case Alpha(A), lower case Alpha(a), upper case Roman(I), lower case Roman(i). The default is number category

UnOrdered lists are referred with the tag called ul type. It has several sub categories in it like disc, square, circle, none. The default is the disc category.





Ques 4. What are called Semantic tags

Semmantic tags/elements do contains the contents with clear information however non-symmentic tags are html tags that do not have a specific meaning. The elements of sementic tags are Header, Footer, Heading, Paragraph.
